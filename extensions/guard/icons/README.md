# Icons Folder

Tempatkan file icon PNG di folder ini dengan ukuran berikut:

- `icon16.png` — 16x16 pixel (icon toolbar kecil)
- `icon48.png` — 48x48 pixel (icon halaman ekstensi)
- `icon128.png` — 128x128 pixel (icon Chrome Web Store)

**Tips:** Gunakan file `public/logo/logo_simply.png` sebagai sumber, lalu resize ke 3 ukuran di atas.
